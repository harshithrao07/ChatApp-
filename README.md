
# ChatApp

A simple Chat Application using SocketIO and React


## Tech Stack

**Client:** React, TailwindCSS, Material-Tailwind

**Server:** Node, Express, SocketIO


## How to run locally

**Client** 
```bash
cd client
npm install 
npm run dev
```
**Server** 
```bash
cd server 
npm install
npm start
```
